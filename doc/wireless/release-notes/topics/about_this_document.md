# About this document
This document provides information about the supported features, release versions, fixed and/or known issues, performance of the Wi-Fi, Bluetooth/802.15.4 radios, including the coexistence with NXP Zephyr downstream release.
The Zephyr release version 4.2.0 has been tested for the wireless SoCs listed below.
## Supported products
-   IW416
-   IW611<sup>1</sup>
-   IW612<sup>1</sup>
-   RW610
-   RW612
-   IW610

[1] Support for IW611 and IW612 in Zephyr release is experimental, intended for evaluation/early development only, not for production.
