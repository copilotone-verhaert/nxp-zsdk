# Introduction

This document explains how to bring up Murata 1XK M.2 Adapter Module \(NXP IW416 based solution\) with Zephyr operating system \(OS\) maintained by NXP in the downstream. Zephyr OS is a lightweight real-time operating system designed for resource-constrained and embedded systems. Zephyr allows for multi-threading, interrupt, memory allocation, inter-thread synchronization, inter-thread data passing, and power management services. To learn more about Zephyr, refer to [the Zephyr Project](https://www.zephyrproject.org/).

**Note:** The setup of the environment and the installation of the NXP Zephyr project are different on Ubuntu and Windows. However, the command-line interface \(CLI\) commands are the same for both.
