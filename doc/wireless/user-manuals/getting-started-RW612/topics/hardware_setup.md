# Hardware setup

This section details the hardware setup required to bring up an RW612 BGA board running Zephyr OS. It includes the required rework and connections.

## Hardware requirements

- RW612 BGA board
- micro-USB to USB-A cable
- Laptop/PC \(check the software setup for OS detail\)

![](../images/hardware-setup.jpg)


## Hardware connection

The following image illustrates the hardware connection required for the setup RW612 BGA board and external laptop/PC.

- Connect the USB cable between the RW612 BGA board's MCU-link interface and the external laptop/PC's USB interface.
- Power on RW612 BGA board.
