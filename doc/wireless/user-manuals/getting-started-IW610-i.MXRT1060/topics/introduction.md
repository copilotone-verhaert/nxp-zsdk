# Introduction

This document explains how to bring up Murata 2LL M.2 Adapter Module (NXP IW610 based solution) with Zephyr operating system (OS) maintained by NXP in the downstream. Zephyr is a lightweight real-time operating system designed for resource-constrained and embedded systems. Zephyr allows for multi-threading, interrupt, memory allocation, inter-thread synchronization, inter-thread data passing, and power management services. To learn more about Zephyr, refer to the [ Zephyr Project](https://www.zephyrproject.org/).

**Note:** The setup of the environment and the installation of the Zephyr SDK are different on Ubuntu and Windows. However, the command-line interface \(CLI\) commands are the same for both.
